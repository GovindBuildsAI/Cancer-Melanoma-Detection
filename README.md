# Lesion Guard - Melanoma Classification

AI-powered melanoma detection demo built with React, TypeScript, Vite, Tailwind CSS, and shadcn-ui.

## Getting Started

Prerequisites:

- Node.js (LTS) and npm installed

Install dependencies:

```sh
npm install
```

Run the development server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

Preview the production build locally:

```sh
npm run preview
```

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn-ui

## Notes

This project includes a placeholder analysis flow. Hook up your own backend or ML service for real predictions.
